## 📦 Installation

### From WordPress.org (Recommended)

1. Go to **Plugins** → **Add New** in your WordPress admin panel
2. Search for "Bootflow Product XML CSV Importer"
3. Click **Install Now** and then **Activate**
4. Navigate to **Bootflow Importer** in your dashboard to start your first import

### Manual Installation

1. Download the plugin from [WordPress.org](https://wordpress.org/plugins/bootflow-product-xml-csv-importer/)
2. Upload the plugin files to the `/wp-content/plugins/bootflow-product-xml-csv-importer` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to the **Bootflow Importer** menu in your dashboard

## 📖 How to Use

1. **Upload File** – Choose your XML or CSV file
2. **Map Fields** – Manually map XML/CSV fields to WooCommerce product fields
3. **Preview** – Check how your data will be imported
4. **Import** – Run the import process with progress tracking
